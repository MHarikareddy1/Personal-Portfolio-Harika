Major Project

My major project titled Simulation of 8-Bit and 16-Bit Asynchronous FIFO. I simulated an asynchronous 8-bit and 16-bit FIFO (First-In, First-Out) which is used to manage data transfer between systems operating at different speeds. The simulation involves using of two clocks: a write clock and a read clock, making it asynchronous. The waveform demonstrates how data is written into the FIFO on each rising edge of the write clock when the write enable signal is active and how it is read out on each rising edge of the read clock when the read enable signal is active. Key flags such as "full" and "empty" indicate when the FIFO cannot accept new data. The simulation shows smooth data flow without overlap between the write and read operations, highlighting the efficiency of the FIFO in handling different clock domains, making it ideal for applications like data buffering and communication systems.
